How to Parse Specific value from multiple elements in single attribute of xml file ?

XML file format

Groovy script to parse value of App_id & sandbox_id & sandbox_name

XmlParser parser = new XmlParser() def xml= readFile "${env.WORKSPACE}/vera.xml" def xmldata1 = parser.parseText(xml) def str1 = xmldata1.attributes().app_id def str2 = xmldata1.attributes().sandbox_name def str3 = xmldata1.attributes().sandbox_id println(str2) println(str3)
